## /ts_flood_open

```txt
/ts_flood_open
```

Ouvre le menu flood. Affiche le niveau actuel, le niveau cible, les modes prédéfinis, et les options pour définir ou forcer un niveau personnalisé.

La même action est enregistrée comme raccourci clavier (`Ouvrir le menu TS Flood`, par défaut : `F1`).
