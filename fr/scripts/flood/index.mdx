TS Flood permet aux admins de controler le niveau d'eau de la map depuis un menu en jeu. Tu peux le monter ou le descendre progressivement, choisir un mode prédéfini, ou forcer un niveau instantanément.

## Fonctionnalités

- Menu zUI en jeu pour gérer le niveau d'eau
- Transitions fluides avec taux configurable
- Modes prédéfinis dans la config
- Saisie de niveau manuel et forçage instantané
- Export client pour lire le niveau actuel
- Support locales EN / FR via ox_lib
