Le script propose des plugins dans `bridge/plugins`, activables ou non via la configuration.

## Activer un plugin

Dans `shared/config.lua`, mettez la valeur correspondante à `true`.

```lua shared/config.lua
Config.Bridge = {
    VehicleDeformation = true,
    -- Autres plugins ici
}
```
