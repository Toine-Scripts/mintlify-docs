Cette ressource permet aux véhicules de rester persistants dans le monde et de rester là où ils ont été laissés.

Elle s'appuie sur **ox_lib** pour lire et appliquer les propriétés des véhicules.

## Fichiers ouverts

Certains fichiers sont « ouverts » : vous pouvez les modifier pour adapter le script à votre serveur. Les fichiers marqués `*` dans l'arborescence ci-dessus sont ouverts.
