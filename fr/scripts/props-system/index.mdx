TS Props System est une ressource complete de gestion de props en jeu pour serveurs FiveM.

Elle permet aux joueurs (ou au staff) de placer, deplacer, modifier et supprimer des props, avec persistance apres redemarrage.

## Fonctionnalites principales

- Menu en jeu pour gerer les props
- Workflow placer, deplacer, changer modele, supprimer
- Spawn de modele personnalise (optionnel)
- Verification de proprietaire pour proteger les actions
- Limites par joueur avec overrides ACE par role
- Persistance SQL et restauration apres restart

## Dependances

- [ts-lib](https://store.toine.me/download/ts-lib)
